IAction Example Usage

This is a provided interface, it is used by the Third Party service to invoke the action to interact with the device, i.e. actually control the actuator, or get the data from a sensor. For an actuator, the action should have a parameter of type Dictionary<String, Object>. 
IDevice screen1 = iDevice;
IDriverService driverService = screen1.getService(DeviceMgmtDriverServiceNames.SCREEN_DRIVER_SERVICE);
IAction ia = driverService.getAction("displayMessage");
Dictionary dic = new Hashtable();
dic.put("message", "Display this message for me please ! ");
ia.invokeAction(dic);
For a sensor, the action has no parameter (null), but returns the sensor data under the Dictionnary format. We assume in this example that the strings "getLightLevel" and "outputLightLevel" has been provided by the driver developer in an ad-hoc documentation. 
IDevice ls1= iDevice;
IDriverService driverService = ls1.getService(DeviceMgmtDriverServiceNames.LIGHT_SENSOR_DRIVER_SERVICE);
IAction ia = driverService.getAction("getLightLevel");
Dictionnary dic = ia.invokeAction(null);
LOG.info("getLightLevel action Return: " + dic.get("outputLightLevel"));
