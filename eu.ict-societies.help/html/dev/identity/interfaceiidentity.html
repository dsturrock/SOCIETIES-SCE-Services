Interface IIdentity

Method Summary
String getBareJid() Returns the bare Jabber ID without resource eg, john@societies.local
String getDomain() Returns the domain of a Jabber ID eg, societies.local from john@societies.local
String getIdentifier() Returns the username of a Jabber ID, eg, John from john@societies.local
String getJid() Returns the full Jabber ID including resource eg, john@societies.local/laptop
IdentityType getType() Returns the CSS Type
