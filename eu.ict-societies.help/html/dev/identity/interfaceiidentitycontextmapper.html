Interface IIdentityContextMapper

Utility class that allows to map context parameters to different identities of the user.
Method Summary
CtxIdentifier addMappedCtxIdentifier(IIdentity publicId, CtxIdentifier attributeID) Add a mapped context parameter.
CtxIdentifier getMappedCtxIdentifier(IIdentity publicId, String attributeType) Retrieve a mapped context parameter.
boolean removeMappedCtxIdentifier(IIdentity publicId, CtxIdentifier ctxIdentifier) Remove a mapped context parameter.
