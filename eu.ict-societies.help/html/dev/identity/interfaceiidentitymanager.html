Interface IIdentityManager

Provides methods for Identity Management
Method Summary
INetworkNode fromFullJid(String jid) Parses a full Jabber ID including the resource to an IIdentity object eg, john@societies.local/laptop
IIdentity fromJid(String jid) Parses a Jabber ID to an IIdentity object, eg, john@societies.local
INetworkNode getIdentityContextMapper() Returns the Domain Authority Node Identity
IIdentityContextMapper getDomainAuthorityNode() Returns the Domain Authority Node Identity
Set<IIdentity> getDomainAuthorityNode() Returns the Domain Authority Node Identity
